in `docs/registers/` (not here) as the authoritative closure document.

Future operational items go to the Enhancement Register or session appendices.

## Why archived

Preserves session-specific evidence and decision rationale that is not carried
forward verbatim in the canonical current-state registers. Per Documentation
Protocol v2, session records are immutable.
